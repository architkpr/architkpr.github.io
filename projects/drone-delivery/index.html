Secure Package Collection for Drone Delivery

During fall of 2019, I enrolled in the Internet of Things course at Columbia University. Towards the latter half of the semester, our team of 3 students decided to develop an IoT system that can be used to securely accept packages delivered using drones and inform the user.

Two challenging problems inspired the development of this project. The first was that of secure landing and delivery of packages when sent by a drone. This is a technical problem being worked on by various groups in the industry. The other, was that of package theft, prevalent in New York City. Several packages in the city get stolen daily, without the owners finding out or alarms being raised.

The project involved using a Raspberry Pi to monitor the mailbox where the package was to be delivered. The Pi communicated (using MQTT) with the drone to inform it about existing packages/obstacles and prepare for a secure landing. The Pi would also send alerts to the user when the package was delivered, and raise an alarm and capture an image to be sent to the user, if the package was picked up without the user’s permission. This required us to write Python scripts and use tools like OpenCV to perform Computer Vision tasks. Our team secured the 2nd position, out of 50 participating teams, in the Annual MS EE Student Project Expo, held that year.
